A Merkle tree is a binary tree that has two main types of nodes:

  • leafs that contain a value (usually block) and it’s hash;
  • nodes that contain a hash of a left and right nodes hashes concatenation; h0123 /
    h01 h23 / \ /
    h0 h1 h2 h3
